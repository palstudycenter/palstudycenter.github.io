Question 2: Find the zeroes of the following polynomials and verify the relationship between the zeroes and the coefficients. (i) x² – 2x – 8

Answer -

Given polynomial:

p(x) = x² – 2x – 8

Factorizing,

x² – 2x – 8 = (x – 4)(x + 2)

Therefore, the zeroes are:

x = 4 and x = –2

Here,

a = 1, b = –2, c = –8

Verification:

Sum of zeroes = 4 + (–2) = 2

–b/a = –(–2)/1 = 2

Hence,

Sum of zeroes = –b/a ✔


Product of zeroes = 4 × (–2) = –8

c/a = –8/1 = –8

Hence,

Product of zeroes = c/a ✔

Therefore, the relationship between the zeroes and the coefficients is verified.

Question 2 (iv): Find the zeroes of the polynomial 6x² – 3 – 7x and verify the relationship between the zeroes and the coefficients.

Answer -

Given polynomial:

p(x) = 6x² – 7x – 3

Factorizing,

6x² – 7x – 3 = (3x + 1)(2x – 3)

Therefore, the zeroes are:

3x + 1 = 0   ⇒   x = –1/3

2x – 3 = 0   ⇒   x = 3/2

Hence, the zeroes are:

x = –1/3 and x = 3/2

Here,

a = 6,   b = –7,   c = –3

Verification:

Sum of zeroes

= (–1/3) + (3/2)

= (–2 + 9)/6

= 7/6

–b/a = –(–7)/6 = 7/6

Hence,

Sum of zeroes = –b/a ✔


Product of zeroes

= (–1/3) × (3/2)

= –1/2

c/a = –3/6 = –1/2

Hence,

Product of zeroes = c/a ✔

Therefore, the relationship between the zeroes and the coefficients is verified.

Question 3: Find the zeroes of the following polynomials, if they exist, by matching the graphs with the given polynomials. (i) p(x) = 2x + 3 (ii) p(x) = x² – 3 (iii) p(x) = 2x² – x – 1 (iv) p(x) = x³ – 4x

Answer -

(i) p(x) = 2x + 3

Putting p(x) = 0,

2x + 3 = 0

2x = –3

x = –3/2

Zero: –3/2

(ii) p(x) = x² – 3

Putting p(x) = 0,

x² – 3 = 0

x² = 3

x = ±√3

Zeroes: –√3, √3

(iii) p(x) = 2x² – x – 1

Factorizing,

2x² – x – 1 = (2x + 1)(x – 1)

Therefore,

2x + 1 = 0 ⇒ x = –1/2

x – 1 = 0 ⇒ x = 1

Zeroes: –1/2, 1

(iv) p(x) = x³ – 4x

Factorizing,

x³ – 4x = x(x² – 4)

= x(x – 2)(x + 2)

Zeroes: –2, 0, 2

These zeroes correspond to the points where the graphs intersect the x-axis.

Question 1: Find the value of k, if x = 1 is a zero of the polynomial p(x) = x² + x + k.

Answer -

Given polynomial:

p(x) = x² + x + k

Since x = 1 is a zero of the polynomial,

p(1) = 0

Substituting x = 1,

(1)² + (1) + k = 0

1 + 1 + k = 0

2 + k = 0

k = –2

Therefore, the value of k is –2.

Question 2: Find the value of k, if x = –2 is a zero of the polynomial 2x² + kx + 4.

Answer -

Given polynomial:

p(x) = 2x² + kx + 4

Since x = –2 is a zero of the polynomial,

p(–2) = 0

Substituting x = –2,

2(–2)² + k(–2) + 4 = 0

2 × 4 – 2k + 4 = 0

8 – 2k + 4 = 0

12 – 2k = 0

2k = 12

k = 6

Therefore, the value of k is 6.

Question 2 (i): Find a quadratic polynomial whose sum of zeroes is \(\frac{1}{4}\) and product of zeroes is \(-1\).

Solution:

Given,

Sum of zeroes (\(\alpha+\beta\)) = \(\frac{1}{4}\)

Product of zeroes (\(\alpha\beta\)) = \(-1\)

The quadratic polynomial whose zeroes are \(\alpha\) and \(\beta\) is:

\[ x^2-(\alpha+\beta)x+\alpha\beta \]

Substituting the given values:

\[ x^2-\frac14x-1 \]

To remove the fraction, multiply the entire polynomial by 4:

\[ 4x^2-x-4 \]

Answer:

\[ \boxed{4x^2-x-4} \]
