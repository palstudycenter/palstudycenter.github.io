# Assets Documentation

## Asset Folder Structure

- `assets/css/` — Project CSS files.
- `assets/js/` — Shared JavaScript files.
- `assets/img/` — Image assets used by the website.
- `assets/vendor/` — Third-party vendor libraries and plugins.

## Shared Styles

- `assets/css/style.css`
  - Main stylesheet for the site.
  - Contains layout, typography, color, and responsive styling for the public pages.

## Shared Scripts

- `assets/js/main.js`
  - Main site script for navigation, scroll behavior, preloader, typed text, and UI interactions.
  - Supports smooth scrolling, active nav highlighting, back-to-top button, mobile nav toggle, and typed hero text.

## Vendor Libraries

The site uses several vendor libraries stored in `assets/vendor/`:

- `aos/` — Animate On Scroll animations.
- `bootstrap/` — Bootstrap CSS and JS for responsive layout and components.
- `bootstrap-icons/` — Icon font library.
- `boxicons/` — Alternate icon set.
- `glightbox/` — Lightbox gallery plugin.
- `isotope-layout/` — Filterable grid layout.
- `php-email-form/` — Form validation helper.
- `purecounter/` — Animated counter module.
- `swiper/` — Slider / carousel plugin.
- `typed.js/` — Typed text effect.
- `waypoints/` — Scroll-driven event triggers.

## Images and Media

- `assets/img/` — Contains the website's image content.
- `assets/files/` — Likely storage for downloadable resources or supporting files.

## Notes

- The project uses CDN-hosted Bootstrap and Google Fonts in addition to local vendor files.
- The vendor folder enables the theme and animations used in the site.
