# Study Center Pages

## Root Pages

- `index.html` — Primary landing page for the study center website.
- `README.md` — Project reference file.

## Public Site Pages in `src/`

- `about.html` — About page for the study center.
- `about2.html` — Alternate about page.
- `contect_us.html` — Contact page for the study center.
- `courses.html` — Courses page listing offered classes or subjects.
- `fees_and_timing.html` — Page for fees, timings, and schedules.
- `student.html` — General student information page.
- `study_material.html` — Page likely showing study material or resources.
- `testimonial.html` — Testimonial page for student reviews.
- `testimonial_2.html` — Alternate testimonial layout.
- `why_pal.html` — Page describing the benefits or features of the study center.

## Signup and Login Flow in `src/signup_and_login/`

- `login_form.html` — Student login page.
- `signup_form.html` — Student signup page.
- `after_login.html` — Post-login landing page for users.
- `home.html` — Additional landing page or home page after signup/login.
- `dashboardscript.js` — Login/signup behavior and redirection script.

## Student Window Pages in `src/student_window/`

- `home.html` — Student window home page.
- `request.html` — Page for requesting a dashboard or service.
