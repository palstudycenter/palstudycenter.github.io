# Study Center Project Overview

## Project Summary

This is a static web project for a study center website built primarily with HTML, CSS, and JavaScript. The site includes a public landing area, student login/signup flow, student dashboard pages, admin pages, and supporting assets such as styles, scripts, images, and third-party vendor libraries.

## Live Site

- https://nayanpal777.github.io/study-center/

## Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery
- Various vendor plugins for UI effects and interactions

## Project Structure

- `index.html`: Main public landing page.
- `admin/`: Admin dashboard pages and admin-specific functionality.
- `assets/`: Shared styles, scripts, image assets, and vendor libraries.
- `src/`: Secondary site pages, login/signup flow, student pages, and additional site content.
- `json_data/`: JSON data used by the project.

## How to Open Locally

1. Open the `d:\study-center` folder in a browser or code editor.
2. Open `index.html` to view the main landing page.
3. Use other HTML files directly in the browser for login, signup, admin, and student pages.

> This project is delivered as a static site, so running a local server is optional but recommended for testing asset paths and browser behavior.
